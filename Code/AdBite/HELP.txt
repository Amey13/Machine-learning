SOFTWARE REQUIREMENTS:
- JAVA 1.6
- Apache Tomcat Server 6.0
- IDE : Eclipse Galileo
- Database : MySQL 5.1
- C++ Compiler: MinGW


INSTALLATION STEPS: 
1) Install MinGW Compiler & add 'C:\MinGW\bin' to system PATH
2) Install MySQL 5.1
3) Load Database into MYSQL from 'Code\Database\adeater2.sql'
4) Deploy ADBite.war on Apache Tomcat server
5) Copy Folder 'Code\SVM' to 'C:\' drive
6) Copy 'Code\SVM\SVM.exe' to JAVA bin folder.
7) Start the Server.
    Open the Browser, & goto http://localhost:8080 (Assuming Server Port is 8080)
